# Sijia Li Portfolio Site

Personal interactive vCard-style portfolio site for analytics, digital commerce, product thinking, and self-introduction.

Live site:

https://sijiali0228.github.io

## Site Purpose

This homepage is designed as a conversation-friendly portfolio rather than a pure project list. It helps explain:

- who Sijia is
- how her statistics background connects to digital analytics
- what she has done in e-commerce and reporting
- how she thinks about customer journeys, product data, and dashboards
- where to find the deeper GitHub analytics evidence

## Main Sections

- Split-screen profile introduction
- Fixed contact and identity panel
- Interactive overview, portfolio, resume, and about sections
- Featured web analytics portfolio
- Work timeline and capability map
- Work style and personality

## Featured Project

[web-analytics-portfolio](https://github.com/SijiaLi0228/web-analytics-portfolio)

This project demonstrates event taxonomy, Matomo tracking planning, SQL KPI modeling, data QA, Power BI dashboard design, and customer journey analysis using a large public e-commerce clickstream dataset.
