## Site Purpose

This homepage is designed as a conversation-friendly portfolio rather than a pure project list. It helps explain:

- who Sijia is
- how her statistics background connects to digital analytics
- what she has done in e-commerce and reporting
- how she thinks about customer journeys, product data, and dashboards
- where to find the deeper GitHub analytics evidence

## Main Sections

- Interactive self-introduction
- Story lenses: data analyst, digital commerce builder, product-minded collaborator, statistics background
- Work timeline
- Featured web analytics portfolio
- Capability map
- Work style and personality
